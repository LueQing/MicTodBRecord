Notes

使用说明

  • 浏览器会优先请求系统当前默认麦克风。
  • 本页面展示的是未校准的相对分贝 dBFS，不是声级计意义上的 dB SPL。
  • TCP 客户端连接 127.0.0.1:7070 后，会收到按行分隔的 JSON 统计。
  • 如果浏览器拒绝权限、麦克风静音或服务断开，页面会直接显示状态。
